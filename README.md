# Windobi
윈도비(windobi)는 데스크탑에서 기동되는 어플리케이션입니다
 - 별도의 백엔드 없이 로컬에서 기동
 - windows에서 기동 확인, mac 및 linux에서의 기동 확인 필요

## 기술
 - Electron
 - React
 - SQLite
 - OpenAI API
 - OpenWeather API
